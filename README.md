# 🔄 Universal Unit Converter

A simple and intuitive Streamlit app that allows users to convert between various units across multiple categories like length, weight, temperature, digital storage, and more — all in real-time.

---

## 📌 Features

- ✅ Converts units for:
  - Length
  - Weight
  - Temperature
  - Area
  - Volume
  - Speed
  - Time
  - Digital Storage
- ✅ Easy-to-use UI with clean Streamlit layout
- ✅ Real-time conversion with `pint` library
- ✅ Error-handling for invalid or unsupported conversions

---

## 🛠️ Requirements

- Python 3.8 or above  
- Libraries:
  - `streamlit`
  - `pint`
